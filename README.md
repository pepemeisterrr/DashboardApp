# Дашборд аналитики

Проект по дисциплине «Углубленное программирование на C#» (3 курс).

## Описание

Desktop-приложение на WPF для визуализации ключевых показателей продаж из базы данных PostgreSQL. Реализована архитектура MVVM, безопасное хранение настроек подключения и интерактивная работа с данными.

## Реализованные функции (согласно ТЗ)

- Подключение к PostgreSQL с шифрованием пароля
- Сохранение настроек подключения в зашифрованном виде
- Фильтрация данных по периоду, категории и региону
- KPI-карточки (общая выручка, количество заказов, средний чек)
- График выручки по категориям (OxyPlot)
- Таблица выручки и заказов по категориям
- Интерактивность: клик по столбцу графика -> фильтрация
- Ручное обновление данных
- Экспорт текущего вида дашборда в PNG

## Технологии

- C# + WPF
- MVVM (CommunityToolkit.Mvvm)
- OxyPlot (графики)
- PostgreSQL + Npgsql
- Шифрование пароля через DPAPI

## Как запустить проект

### 1. Подготовка базы данных

1. Установите PostgreSQL (версия 12 или выше).
2. Откройте pgAdmin или любой другой клиент.
3. Выполните скрипт DashboardApp/ `CreateDatabase.sql`  (он создаст таблицы и заполнит тестовыми данными).

### 2. Запуск приложения

1. Откройте решение `DashboardApp.sln` в Visual Studio.
2. Запустите проект.
3. В меню Настройки выберите Подключение к БД...
4. Введите данные вашего PostgreSQL:
   - Host: `localhost`
   - Port: `5432`
   - Database: `dashboard_db`
   - Username и Password
5. Нажмите **"Проверить соединение"**, а затем **"Сохранить"**.

После этого приложение загрузит данные и отобразит дашборд.
